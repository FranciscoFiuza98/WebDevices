O que é a WebDevicess?

A WebDevices é uma aplicação web que tem como objetivo facilitar a manipulação de vários dispositivos, nomeadamente sensores, atuadores e dispositivos de display, quer seja para uma empresa ou até mesmo uma escola. Este projeto foi desenvolvido no âmbito da unidade curricular de Projeto II da licenciatura de Tecnologias e Sistemas de Informação para a Web na Escola Superior de Media Artes e Design, paralelamente com as unidades curriculares de Engenharia de Software e Programação Web II.

Funções

Funções da aplicação WebDevices

Adicionar

Esta função irá permitir-lhe adicionar dispositivos á sua conta, tais como, sensores, atuadores e dispositivos de display.

Consultar

Aqui poderá consultar as informações dadas pelos dispositivos montados anteriormente e removêr-lo.

Monitorizar

Por fim, irá puder, com base no dispositivo, monitorizá-lo, isto é, como por exemplo, ligá-lo, medir temperaturas, definir a resolução do display e muito mais.
